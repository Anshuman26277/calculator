Simple Calculator

This project is a simple calculator built with Python's Tkinter library. It supports basic arithmetic operations like addition, subtraction, multiplication, and division.

How to Run

  1. Ensure Python 3.x is installed on your system.
  2. Save the script as simple_calculator.py.
  3. Run the script using: 
    python simple_calculator.py
  4. Use the on-screen buttons to perform calculations.
